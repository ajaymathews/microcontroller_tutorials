TOUCHGFX_WORKING METHOD
--------------------------->

1)CRETE A LAYOUT IN TOUCHGFX
2)CLICK THE GENERATE CODE
3)OPEN THE CUBEIDE AND CLICK OPEN PROJECT AND CHOOSE THE GENERATED CODE FOLDER INSIDE NAMED CUBEIDE,
  (BETTER OPTION IS THAT YOU OPEN FOLDER NAMED CUBEIDE USING FILE MANAGER AND CLICK THE .PROJECT FILE IN FOLDER 
   CHOOSE THE STM32F429-DISCO AND DO THE BELOW PROCEDURE)
4)CLICK ON THE MAIN PROJECT NAME(STM32F429-DISCO) AND RIGHT CLICK AND CHOOSE DEBUG,
  MAKE SURE NOTHING SHOWS ERROR
5)THEN RUN THE PROGRAM.THIS WILL DOWNLOAD THE CODE INTO THE BOARD.


HARDWARE-INTERATION
------------------------>

6)IF YOU WANT TO INCLUDE  SOME HARDWARE INTEGRATON INTO THIS,THEN CREATE THE VIRTUAL FUNCTION IN THE INETRACTION PART OF DESIRED BUTTON OR INTERFACE
  WITH ANY NAME YOULIKE , THAT WOULD BECOME THE FUNCTION NAME.
7)OPEN ANOTHER IDE/MX AND GENERTE THE HARDWAE CODE YOU NEEDED LIKE LED,UART ETC.., AFTER THAT GO TO THE MAIN
  IN THE GENERTAED CODE, AND COPY ALL THE INIT() FUNCTIONS IN THE MAIN , WHICH INCLUDES CLOCK CONFIGURATION AND THE PERIFERRAL INITS().
8)THEN FROM THE  APPLICATION/USER/GENERATED GO TO THE SCREE1VIEWBASE.CPP AND IN THE BUTTONHANDLER, OR SIMILAR
  ACTION, WE COULD FIND THE VIRTUAL FUNCTION NAME WE GIVEN.CHECK ITS THE SAME AS THE GIVEN ONE.
9)THEN GO TO APPLICATION/USER/GUI/ IN THAT THERE IS A SCREENVIEW.CPP AT THE TOP INCLUDE "stm32f4xx_hal.h" AND PASTE THE INIT FUNCTIONS COPIED.
  FIND FUNCTION DEFENTIONS LOOKS LIKE 'SCREENVIEW::XXXX'(MOST OF THEM ARE LIKE THAT) COPY IT AND THEN PASTE AS ANOTHER FUN, BUT CHNGE THE FUN
  WITH OUR VIRTUAL FUN,THEN WRITE THE DESIRED HARDWARE CODE YOU NEEDED LIKE BLINKING LEDS INTO THAT FUNCTION.
10)GO TO TOP OF THE SAME PGM AND CLICK ON HEADER AND CLICK OPEN FILE CALLED SCREENVIEW.HPP AND IN THAT TYPE OUR VIRTUAL FUNCTION 
  DECLARATION INTO THAT SAME AS THE DECLARATION OF OTHER FUNCTIONS IN THE HEADER. 

 * COMPILE AND RUN
